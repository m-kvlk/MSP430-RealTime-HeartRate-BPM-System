# MSP430 Real-Time Heart Rate (BPM) Measurement System

This project implements a real-time heart rate (BPM) measurement system using an MSP430 microcontroller and an analog pulse sensor.

---

## Project Overview

- Analog signal sampling via ADC
- Hardware (RC) and software filtering
- Timer interrupt-based period detection
- Real-time BPM calculation
- UART communication with PC
- Live BPM visualization using Python
